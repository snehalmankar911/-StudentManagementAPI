📌 Student Management System – ASP.NET Core Web API
🚀 Project Overview

This is a Student Management System API built using ASP.NET Core Web API, SQL Server, and JWT Authentication.
It supports full CRUD operations with secure endpoints and clean layered architecture.

⚙️ Technologies Used
ASP.NET Core Web API
Entity Framework Core
SQL Server
JWT Authentication
Swagger
C#
🏗️ Architecture
Controller Layer
Service Layer
Repository Layer
Middleware (Exception Handling)
Database Layer (EF Core)
📊 Features
Add Student
Get All Students
Update Student
Delete Student
JWT Authentication
Global Exception Handling
Swagger API Documentation
